### Чтобы выйти из аккаунта GitHub, необходимо:

1. Навести курсор на аватар, чтобы появилось **меню**.
<img width=450 src="https://i.ibb.co/78F2TMK/Screenshot-1.png" />

2. Найти кнопку **"Sign Out"** и нажать на неё.

<img width=350 src="https://i.ibb.co/6RVBcKnd/Screenshot-2.png" />

3. Нажать на **большую красную кнопку** в центре экрана для выхода из аккаунта.

<img src="https://i.ibb.co/8D7mRPqq/Screenshot-3.png" />
